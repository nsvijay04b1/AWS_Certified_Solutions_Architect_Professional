# AWS Certified Solutions Architect – Professional (SAP-C01) Exam Guide

* Exam format:
        
      The exam may contain two types of questions:
        Multiple choice
        Multiple response
      Minimum passing score: 750/1000
      Number of questions: 75
      Time: 190 minutes to complete the exam (with a possibility to request 30 minutes extra for non-native english speakers)

* Official exam guide  [link](https://d1.awsstatic.com/training-and-certification/docs-sa-pro/AWS-Certified-Solutions-Architect-Professional_Exam-Guide.pdf)

* The percentage in each domain represents only scored content

    
    | Domain | % of Exam | 
    |--------|-----------|
    | Domain 1: Design for Organizational Complexity | 12.5% |
    | Domain 2: Design for New Solutions | 31% |
    | Domain 3: Migration Planning | 15% |
    | Domain 4: Cost Control | 12.5% |
    | Domain 5: Continuous Improvement | 29% |
    | TOTAL | 100% |
